# Product Naming: Cosmio

## Naming Strategy (Diamond Framework)

| 항목 | 내용 |
|------|------|
| 목표 | 우주의 광활함과 신비로움을 한 단어로 환기, "GitHub contribution 시각화 도구"라는 정체성이 자연스럽게 연상 |
| 강점 | 예술적 우주 테마, 4가지 뷰, 애니메이션 SVG, 오픈소스 |
| 핵심 메시지 | "당신의 코딩 기록이 우주가 된다" |
| 브랜드 성격 | 신비로운, 경외감을 주는, 세련된 |
| 사운드 | 부드럽고 깊은 울림 (L, N, M, S) |

## Final Decision

### Cosmio

- **어원**: Cosmos(우주) + -io(모던 tech suffix)
- **발음**: 코스미오 (3음절, 자연스러운 흐름)
- **인상**: 부드럽고 따뜻한 광활함. 친근하면서 모던.

### 선정 이유

1. **확장성** — "Cosmos" 자체를 품고 있어 어떤 우주 테마가 추가되어도 자연스러움
2. **가용성** — npm, GitHub 모두 사용 가능
3. **기억 용이성** — 직관적 발음, 들으면 바로 쓸 수 있음
4. **브랜드 적합성** — 고급스러우면서도 접근하기 쉬운 톤

## Runner-up Candidates

| 이름 | 어원 | 인상 | 미선정 이유 |
|------|------|------|------------|
| Stellex | Stella(별) + ex(확장) | 차갑고 날카로운 신비감 | 기술 도구 느낌이 강해 예술적 감성이 약함 |
| Orbitra | Orbit(궤도) + ra(울림) | 역동적이고 장대한 궤도 | 좋지만 Cosmio보다 테마 확장성이 제한적 |

## Usage

- npm package: `cosmio`
- GitHub repo: `cosmio`
- GitHub Action: `cosmio-action` 또는 `cosmio`
- CLI command: `npx cosmio`
